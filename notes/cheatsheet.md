# My command cheat-sheet

  ## Looking around
  - pwd  = where am I?
  - ls   = what's here?  (ls -a shows hidden files too)
  - cat  = show what's inside a file
  - cd foldername = go into a folder
  - cd .. = go up one level
  - cd ~  = go home

  ## Editing
  - nano filename = open (or create) a file to edit
    - Ctrl+O then Enter = save
    - Ctrl+X = exit

  ## Saving my work (the loop)
  - git status = what changed?
  - git diff   = what lines changed?
  - git add .  = prepare the snapshot
  - git commit -m "note" = save the snapshot
  - git push   = upload to GitHub
  - git log    = see my history (press q to quit) - can also use arrow keys
